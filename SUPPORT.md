# Support

Thanks for using `remark-lint-frontmatter-validation`.

## How to get help

1. Check project docs first:
   - [README.md](./README.md)
2. Search existing issues:
   - <https://github.com/Nick2bad4u/remark-lint-frontmatter-validation/issues>
3. Open a new issue if needed:
   - <https://github.com/Nick2bad4u/remark-lint-frontmatter-validation/issues/new>

## What to include in an issue

Please include:

- Node.js version
- TypeScript version
- remark version
- `remark-lint-frontmatter-validation` version
- Minimal reproduction snippet
- Expected behavior and actual behavior

If possible, include your remark config snippet and the smallest schema/frontmatter
pair that reproduces the issue.

## Support scope

This repository supports:

- Bug reports in plugin, CLI, and schema resolution behavior
- False positives and false negatives in validation findings
- Option and docs improvements
- Release and packaging issues for this package

This repository does not provide private support channels.

## Security reports

For security concerns, do not open a public issue.
Follow [SECURITY.md](./SECURITY.md).
